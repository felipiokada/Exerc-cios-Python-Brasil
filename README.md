# Exercícios-Python-Brasil
Pasta de exercícios do site Python Brasil feitos em Java

Esta pasta tem como objetivo desenvolver habilidades e competências na linguagem Java realizando
exercícios do site Python Brasil na IDE Eclipse. Até o fim das atividades do site, o programador DEVE
fazer anotações sobre as melhores práticas de programação e, caso necessário, registrar informações
com relevância técnica da própria linguagem para fins de consulta.
